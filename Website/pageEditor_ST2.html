Home Page

Technical Description

The Home Page serves as the main introduction to the website. It includes the following features:

  • HTML: Used semantic tags like <header>, <main>, and <section> to structure the content.
  • CSS: Applied a background image with background-image and used rgba for semi-transparent backgrounds. Flexbox was used for layout.
  • Accessibility: Added alt text for images and ensured proper contrast for text readability.

Accessibility

Accessibility was addressed by:

  • Using alt text for all images.
  • Ensuring proper contrast ratios for text and background colors.
  • Making the navigation bar keyboard-friendly.

Template + Navigation

Technical Description

The Template and Navigation were designed to ensure consistency across the website. Key features include:

  • HTML: Created a reusable template with <header>, <nav>, <main>, and <footer>.
  • CSS: Used a global CSS file (gstyles.css) to define styles for the header, navigation, and footer. Added hover effects and active page highlighting.
  • Accessibility: Ensured keyboard navigation and screen reader compatibility.

Accessibility

Accessibility was addressed by:

  • Adding aria-label attributes for interactive elements.
  • Ensuring the navigation bar is keyboard-friendly.

Table Page

Technical Description

The Table Page displays education statistics in a structured and interactive way. Key features include:

  • HTML: Used <table>, <thead>, <tbody>, and <tr> tags. Merged cells using rowspan and colspan.
  • CSS: Applied zebra striping with nth-child, hover effects, and shadows for interactivity.
  • Accessibility: Added aria-label attributes and tooltips for better accessibility.

Accessibility

Accessibility was addressed by:

  • Adding aria-label attributes for table headers.
  • Ensuring proper contrast ratios for text and background colors.

Content Page

Technical Description

The Content Page provides information about learning strategies. Key features include:

  • HTML: Used semantic tags like <section> and <nav> for internal navigation. Added images with alt text for accessibility.
  • CSS: Applied responsive design using flexbox for the internal navigation. Used box-shadow and border-radius for a modern look.
  • Accessibility: Added alt text for images and ensured proper contrast for text readability.

Accessibility

Accessibility was addressed by:

  • Using alt text for all images.
  • Ensuring proper contrast ratios for text and background colors.
  • Making the internal navigation keyboard-friendly.

Challenges and Lessons Learned

Challenge 1: Ensuring the navigation bar was responsive and accessible. I resolved this by testing with screen readers and adding ARIA attributes.

Challenge 2: Making the table page interactive without compromising accessibility. I learned how to use CSS hover effects and ARIA labels effectively.

Lesson Learned: Accessibility is crucial in web design, and semantic HTML plays a key role in achieving it.

Compliance

The pages I developed comply with Janet regulations. This includes:
